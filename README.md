# salarycalculatorfordeductedtax
This app calculates the annual salary of a salaried individual in Pakistan for the tax year 2025. This can be used to correctly file income tax return of a salaried individual. The tax payer need to input total payable tax in exact amount i.e 1000.25 including the integral part. This figure is available on the salary slip of june 2025.
INPUT: TOTAL PAYABLE TAX FROM THE SALARY SLIP OF JUNE 2025
YOU WILL NEED PYTHON TO RUN THIS
